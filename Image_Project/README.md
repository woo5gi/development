### 횐경설정

#### 서버
npm init -y
npm i express
npm i -D nodemon
npm i multer
npm i uuid mime-types
npm i bcryptjs

#### 클라이언트
npx create-react-app client
npm i axios
npm i react-toastify
npm i mongoose
npm i dotenv
npm i react-router
npm i react-router-dom

#### 시작
- 서버 : 

cd server
npm run dev

- 클라이언트 : 

cd client
npm run start
